# Gameflix

Projeto tenta criar um simulacro de Netflix, mas sobre jogos.

## Categorias
- Aclamados pela critica
- Jogos por genero:
    - Action;
    - Indie;
    - Adventure;
    - RPG;
    - Shooter;
    - Racing
    - Fighting
