# Gameflix

Projeto tenta criar um simulacro de Netflix, mas sobre jogos. Nesse projeto foi usado a api [RAWG](https://api.rawg.io/docs/).

## Categorias
- Aclamados pela critica
- Jogos por gênero:
    - Action;
    - Indie;
    - Adventure;
    - RPG;
    - Shooter;
    - Racing
    - Fighting
